# Dashboard Data (React + TypeScript) 
Mini-Projet vitrine, orienté data-viz
Objectif : Démontrer une architecture front claire, et un usage propre de TS. 

# Stack Technique : 
- React 
- Node (22)
- Typescript
- Vite

# Lancer en local : 
```bash
npm install
npm run dev 
```

# Fonctionnalités
Dashboard basique avec graphiques + filtres
Table "Top items"

Données via mock API (front) lors de la première phase
